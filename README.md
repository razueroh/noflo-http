# HTTP for NoFlo
[![Build Status](https://secure.travis-ci.org/razueroh/noflo-http.png?branch=master)](http://travis-ci.org/razueroh/noflo-http)

This package provides utility components to create HTTP servers and clients in the NoFlo Node.js runtime. Based on [HTTP Node.js](https://nodejs.org/api/http.html).

## Package Installation

    npm install noflo-http --save
